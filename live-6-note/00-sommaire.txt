Live 6 - React Native
Dans ce sixième live, nous allons voir ensemble les hooks permettant de stocker un state dans un composant, 
partager un state entre plusieurs composants, exécuter une opération suite au chargement du composant.
Pour cela, nous allons voir ensemble les sujets suivants :

useState
useEffect
useContext / createContext
useReducer
